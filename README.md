# console-app
First console app created with nodejs.

# options 

``` 
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Número del cual quieres obtener la tabla de multiplicar.
                                                             [number] [required]
  -l, --listar   Te muestra la tabla de multiplicar completa en la consola.
                                                                       [boolean]
  -h, --hasta    Te hasta qué número quieres que se haga la multiplicación.
                                                          [number] [default: 10]
```
